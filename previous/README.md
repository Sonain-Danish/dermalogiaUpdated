<p align="center">A Nest JS Application for Fixomate Api by Team Chainspair</p>

## Getting Started

# Install Packages

```bash
$ yarn install
```

# Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn build
$ yarn start:prod
```

- Next JS
